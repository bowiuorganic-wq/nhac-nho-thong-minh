# Smart Reminder Web

Bản web test của Smart Reminder.

## Chạy local

```bash
npm install
npm start
```

Mở: http://localhost:5173

## Deploy Render

- Build Command: `npm install`
- Start Command: `npm start`
- Root Directory: để trống nếu `package.json` nằm ở thư mục gốc repo.

## Lưu ý

Bản web dùng để test giao diện, logic rule, upload quy ước Google Sheet, upload nhạc chuông và trải nghiệm trên Chrome.
Báo thức khi đóng tab/khóa màn hình không ổn định như bản Android native.
